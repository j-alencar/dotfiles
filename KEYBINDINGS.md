# Shell keybindings

zsh emacs mode (`bindkey -e`), with atuin replacing history search and WezTerm Alt passthrough enabled.

## Line editing

| Key | Action |
|-----|--------|
| `Ctrl+A` | Beginning of line |
| `Ctrl+E` | End of line |
| `Alt+F` | Forward word |
| `Alt+B` | Backward word |
| `Alt+D` | Delete word forward |
| `Ctrl+W` | Delete word backward |
| `Ctrl+K` | Kill to end of line |
| `Ctrl+U` | Kill whole line |
| `Ctrl+Y` | Yank (paste killed text) |
| `Ctrl+T` | Transpose characters |
| `Alt+T` | Transpose words |
| `Alt+U` | Uppercase word |
| `Alt+L` | Lowercase word |
| `Alt+.` | Insert last argument of previous command |

## History

| Key | Action |
|-----|--------|
| `Ctrl+R` | Atuin fuzzy history search |
| `↑` | Atuin prefix history search |

## Completion

| Key | Action |
|-----|--------|
| `Tab` | Complete / open menu |
| `Shift+Tab` | Reverse through menu |
| `Ctrl+G` | Cancel completion |

## WezTerm

| Key | Action |
|-----|--------|
| `Ctrl+Q` | Toggle fullscreen |
| `Ctrl+'` | Clear scrollback |
| `Shift+PageUp/Down` | Scroll page |
| `Ctrl+Shift+T` | New tab (Windows) |
| `Ctrl+Shift+W` | Close tab (Windows) |
| `Ctrl+Tab` | Next tab (Windows) |
